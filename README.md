# REFLECTION

1. I implemented OOP principles and TypeScript features by using classes and subclasses, using async/await functions to handle promises, and by using the fetch function to interact with the API.

2. I encountered many challenges during this project because TypeScript is new to me, but the biggest challenge I faced was tying everything together. I feel like I understand each feature when it's introduced, but when we have to put everything together, like in this project, I get a little confused. To overcome this challenge I had to go back and re-read the lessons to get a better understanding of how everything connects. I also had to google anything I still didn't understand and try to map out the process. 

3. I handled asynchronous operations using the async/await functions with a try/catch block to handle any error that may occur. I also used a customError module to create errors for different error types to give more specific feedback.
